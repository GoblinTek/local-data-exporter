\# Local Data Exporter



Local Data Exporter is a RuneLite plugin that writes local account snapshots to JSON files.



\## Features



\- Exports skill levels and XP

\- Exports bank, inventory, and equipment contents

\- Exports Grand Exchange offer data

\- Exports quest states

\- Exports achievement diary completion

\- Exports basic status, location, world, and account metadata

\- Caches last known container data when bank, inventory, or equipment data is temporarily unavailable

\- Supports configurable export directory and export interval



\## Output



By default, JSON files are written to:



```

.runelite/local-data-exporter

```
